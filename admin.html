About

Content Management System — Indiabulls Securities Support Portal

Version 2.0.0 · Built for the Support Admin Team

This dashboard reads static articles from faq.html and all dynamic articles from DynamoDB via the cloud API. All writes (add, edit, status change, delete) go directly to DynamoDB — no localStorage is used in the content pipeline.

Session keys (browser only): ibulls_admin_auth, ibulls_admin_pass — used for local admin login session only, not content storage.
